# Plan With Learnings

When editing plan sections in `tasks/current.md`:

- Before drafting or revising the `## Plan` section, check `docs/learnings/`
  for lessons whose `category` or `tags` are relevant to the current task.
- If applicable learnings exist, reference them in the plan
  (e.g., "See docs/learnings/2025-12-15-caching-pitfall.md").
- Do not copy entire lessons into the plan. A brief summary and a file
  reference are sufficient.
- If no relevant learnings exist, proceed normally without adding
  artificial references.
